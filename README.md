# Financial-Research-System
An automated fundamental research pipeline and valuation engine. Built with Python and Streamlit to integrate NLP market signals, forensic accounting, and capital velocity modeling for institutional equity analysis


# Key Features
- **Automated DCF Modeling:** End-to-end valuation for 500+ global tickers.
- **NLP Signal Detection:** Integrated sentiment analysis to quantify the impact of financial news on growth assumptions.
- **M&A Velocity Tracking:** Dedicated logic to analyze inorganic growth engines and synergy potential.
- **Institutional Reporting:** One-click generation of professional PDF research dossiers.

# Tech Stack
- **Languages:** Python (Pandas, NumPy, Plotly)
- **Data Sourcing:** YFinance API integration
- **Deployment:** Streamlit Cloud
- **Reporting:** FPDF Engine
